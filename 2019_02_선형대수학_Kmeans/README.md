# K-means Clustering
주어진 데이터를 군집화

## 목표
주어진 Matrix 및 title list, word list를 입력 받은 후 K-mean Clustering을 통하여 주어진 자료를 군집화,<br>
그 과정을 추적하여 변화하는 J-Clust 값과 각 클러스터를 대표하는 5개의 문서 제목을 확인한다.<br>

## 환경
언어 : C<br>
